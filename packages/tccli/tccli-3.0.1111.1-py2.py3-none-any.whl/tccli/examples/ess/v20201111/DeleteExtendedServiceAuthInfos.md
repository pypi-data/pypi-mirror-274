**Example 1: 删除企业扩展服务授权- 自动签**



Input: 

```
tccli ess DeleteExtendedServiceAuthInfos --cli-unfold-argument  \
    --Operator.UserId abc \
    --ExtendServiceType OPEN_SERVER_SIGN \
    --UserIds abc
```

```
{

        "RequestId": "s1697xxxxxxxxx107"

}
```
