**Example 1: 失败示例**

请求参数不正确导致返回错误

Input: 

```
tccli partners DescribeAgentClients --cli-unfold-argument  \
    --Offset 0 \
    --Limit 0
```

Output: 
```
{
    "Response": {
        "RequestId": "3c140219-cfe9-470e-b241-907877d6fb03",
        "Error": {
            "Code": "InvalidParameter",
            "Message": "Limit:0 is not int!"
        }
    }
}
```
