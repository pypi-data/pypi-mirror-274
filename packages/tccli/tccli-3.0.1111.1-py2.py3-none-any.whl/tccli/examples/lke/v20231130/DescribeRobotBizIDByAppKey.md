**Example 1: 通过AppKey查询机器人业务ID**

通过AppKey查询机器人业务ID

Input: 

```
tccli lke DescribeRobotBizIDByAppKey --cli-unfold-argument  \
    --AppKey xITNfdVe
```

Output: 
```
{
    "Response": {
        "BotBizId": "1751945013419311104",
        "RequestId": "ad3d5826-6cda-4f3c-a620-1749608b7aff"
    }
}
```
