**Example 1: 查询账号所有可授予权限**

查询账号所有可授予权限

Input: 

```
tccli cynosdb DescribeAccountAllGrantPrivileges --cli-unfold-argument  \
    --Account.Host % \
    --Account.AccountName test \
    --ClusterId cynosdbmysql-xxxxxxxx
```
