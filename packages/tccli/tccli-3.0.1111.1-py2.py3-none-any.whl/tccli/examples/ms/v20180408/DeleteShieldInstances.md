**Example 1: 根据ItemId对加固实例进行删除**

通过传入ItemId集合删除1个或者多个加固实例

Input: 

```
tccli ms DeleteShieldInstances --cli-unfold-argument  \
    --ItemIds xyuu-csu-ee78236l hhussxu-hui2677-kk
```

Output: 
```
{
    "Response": {
        "RequestId": "5e93a212-ca01-0fdc-eedd-5a1fce5e83e6",
        "Progress": 1
    }
}
```
