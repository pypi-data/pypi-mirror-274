**Example 1: 开启告警**

开启告警

Input: 

```
tccli wedata ModifyBaselineAlarmStatus --cli-unfold-argument  \
    --IsAlarm true \
    --Id 24 \
    --ProjectId 1464962169590902784
```

Output: 
```
{
    "Response": {
        "Data": true,
        "RequestId": "34ef7381-4ccc-473d-b6da-81827752cde2"
    }
}
```
