**Example 1: 获取基础防护状态**

获取基础防护状态

Input: 

```
tccli antiddos DescribeBasicDeviceStatus --cli-unfold-argument  \
    --IpList abc \
    --IdList abc \
    --FilterRegion 1
```

Output: 
```
{
    "Response": {
        "Data": [
            {
                "Key": "abc",
                "Value": "abc"
            }
        ],
        "CLBData": [
            {
                "Key": "abc",
                "Value": "abc"
            }
        ],
        "CnameWafData": [
            {
                "Key": "1.1.1.1",
                "Value": "1"
            }
        ],
        "RequestId": "abc"
    }
}
```
