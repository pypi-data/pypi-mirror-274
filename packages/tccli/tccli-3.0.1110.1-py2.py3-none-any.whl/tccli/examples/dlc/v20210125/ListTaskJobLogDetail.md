**Example 1: 查询日志详情**

查询spark 作业任务的日志详情

Input: 

```
tccli dlc ListTaskJobLogDetail --cli-unfold-argument  \
    --TaskId abc \
    --StartTime 0 \
    --EndTime 0 \
    --Limit 0 \
    --Context abc \
    --Asc True \
    --Filters.0.Name abc \
    --Filters.0.Values abc \
    --BatchId abc
```

Output: 
```
{
    "Response": {
        "Context": "abc",
        "ListOver": true,
        "Results": [
            {
                "Time": 0,
                "TopicId": "abc",
                "TopicName": "abc",
                "LogJson": "abc",
                "PkgLogId": "abc"
            }
        ],
        "LogUrl": "abc",
        "RequestId": "abc"
    }
}
```
